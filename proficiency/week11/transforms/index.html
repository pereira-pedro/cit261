Designing, Defining, and Triggering CSS3 Transforms without Custom Libraries (Thought Library)

  • CSS3 Transforms

    CSS transforms is to change an element from one state to another. With the CSS transforms property, you can rotate, move, skew, and scale elements. There are 2D and 3D transforms. Transforms are triggered when an element changes states, such as on mouse-hover or mouse-click.

    These are the 2D CSS functions available:

    • translate()
    • rotate()
    • scaleX()
    • scaleY()
    • scale()
    • skewX()
    • skewY()
    • skew()
    • matrix()

    Each transform function has its own parameters. The most special one is the matrix(). It accepts 6 parameters being a subset of functions above. Although very powerful, it is hard to use manually.

    matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())
