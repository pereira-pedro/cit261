HTML5 Tags - Video, Audio, and Canvas

  • Overview

    HTML5 introduces new tags to accommodate the increasingly interactive and multimedia nature of the Web. While there have been numerous ways to embed video, audio, and dynamic imagery in the past, the new web standard attempts to make this easier, more consistent, and more reliable.

    While the HTML tag to embed multimedia content is standardized, supporting media types and formats is browser dependent. More information can be found in https://developer.mozilla.org/en-US/docs/Web/Media/Formats

    Canvas
    
      
    The HTML5 canvas tag is used to draw graphics, on the fly, via
        scripting (usually JavaScript).
        However, the canvas element has no drawing abilities of its own (it is only a container for
        graphics) - you must use a script to actually draw the graphics.
    

      
    function draw(canvas, points) {
  var ctx = canvas.getContext("2d");
  ctx.beginPath();
  ctx.color = '#ff0000';
  var pastRow = null;
  const rect = canvas.getBoundingClientRect();

  points.forEach(function (row) {
     if (pastRow !== null) {
      ctx.moveTo(pastRow.x - rect.left, pastRow.y - rect.top);
      ctx.lineTo(row.x - rect.left, row.y - rect.top);
      ctx.stroke();
    }

    pastRow = { identifier: row.identifier, x: row.x, y: row.y };
});
