  • 1 - AJAX

    Definitions

    • JavaScript Object Notation
    • It's a lightweight format for storing and transporting data
    • Often used when data is transferred from/to server
    • It’s easy to understand and use

    Syntax

    • A JSON object in enclosed between { } (curly braces)
    • An array of objects are enclosed between [ ] (square braces)
    • Property name and its value are separated by : (colon)
    • Each tuple (name, value) is separated by , (comma)

    Convert String to JSON

    • It’s easy to convert from string to JSON In Javascript
    • Create a string with the JSON syntax
    • The use the built-in function JSON.parse()to convert it

    Example

    var buffer = '';
const students = JSON.parse("{ [ { name: 'Pedro Pereira', email: 'per17009@byui.edu', gpa: 3.6}, { name: 'Tatiana Pereira', email: 'tmarin@byui.edu', gpa: 4}] }");
